# commerce-option-matrix

대규모 커머스 환경에서 발생하는 **상품 옵션 조합(SKU Matrix)** 문제를 해결하기 위한  
Vue 3 + TypeScript 기반 **옵션 구성 및 SKU 생성 UI** 프로젝트입니다.

스마트스토어와 유사한 판매자 옵션 입력 화면을 재현하며,  
옵션 개수 증가로 인한 **조합 폭증, 데이터 정합성, 운영 UX 문제**를 구조적으로 해결하는 데 초점을 맞췄습니다.

---

## 📌 프로젝트 배경

커머스 상품 옵션은 단순 입력 문제가 아니라,

- 옵션 개수 증가에 따른 **조합 수의 기하급수적 증가**
- 옵션 변경 시 기존 SKU 데이터(가격/재고/사용여부) **보존 문제**
- 대량 SKU 생성 시 **UI 성능 저하 및 운영 리스크**

등을 동반하는 **고카디널리티(high-cardinality) 문제**입니다.

본 프로젝트는 이러한 문제를 **프론트엔드 단에서 어떻게 안전하게 통제할 수 있는지**를 보여줍니다.

---

## ✨ 주요 기능

### 1. 옵션 구성 (최대 7개)
- 옵션명/옵션값을 동적으로 추가·삭제
- 옵션값 입력 방식
  - 콤마(,) 또는 줄바꿈 입력 지원
  - 중복 값 자동 제거
- 옵션별 파싱 결과 즉시 표시

### 2. 옵션 조합(SKU) 생성
- 옵션값들의 Cartesian Product 기반 SKU 생성
- 조합 수 사전 계산 후 상한 초과 시 생성 차단
- 기본 조합 상한: **5,000**

### 3. SKU 데이터 보존
- 옵션 재적용 시에도 기존 SKU의
  - 옵션가
  - 재고수량
  - 사용여부
  - 관리코드  
  를 **stable key 기반으로 유지**

### 4. SKU 테이블 관리
- SKU 목록 테이블 렌더링
- 판매상태 자동 계산
  - 재고 > 0 && 사용여부 = Y → 판매중
  - 그 외 → 품절
- 개별 삭제 / 선택 삭제 지원

### 5. 선택 SKU 일괄 수정
- 옵션가 / 재고수량 / 사용여부 일괄 적용
- 운영자 UX 고려
